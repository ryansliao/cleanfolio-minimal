Projects

Activity Based Model w/ SANDAG

Created machine learning alternatives to SANDAG’s proprietary Activity-Based Model. Our models predicted vehicle type, their fuel type, and their age using SANDAG provided household data. Most importantly, we were able to predict county-wide trip destinations using SANDAG's synthetic population and census data using a parameter-tuned decision tree classifier with an accuracy and F1 of ~72% predicting over 240 classes while decreasing runtime by ~90%.

Clothing Size Recommender System

Fitted an unsupervised learning algorithm to predict a user’s most comfortable size based on clothing reviews from RentTheRunway. We identified the most predictive features (weight, height, age, bust, cup) and tested a variety of linear and logistic regressions using various summary statistics, with the final XGBoost regression producing a mean-squared error of 17.91.
